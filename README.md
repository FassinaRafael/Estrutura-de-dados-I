# Estrutura de Dados I

Este repositório contém os códigos desenvolvidos durante a disciplina de Estrutura de Dados I.

## Conteúdo

- **Array_Strings.c**: Demonstra o uso de arrays de strings.
- **Array_Structs.c**: Demonstra o uso de arrays de structs.
- **ArvoreBinariaV1.c**: Implementação de uma árvore binária com travessias pré-ordem, em-ordem e pós-ordem.
- **busca_Struct.c**: Implementa a busca em um array de structs por matrícula e por nome.
- **busca_binaria.c**: Implementação do algoritmo de busca binária.
- **fila.c**: Implementação de uma fila com as operações `enqueue`, `dequeue`, `peek` e `imprime`.
- **lista_encadeadaV1.c**: Implementação de uma lista encadeada com várias operações, como inserção, remoção, busca, concatenação, etc.
- **merge_sort.c**: Implementação do algoritmo de ordenação merge sort.
- **ordenacao_basica.c**: Implementa os algoritmos de ordenação bubble sort, selection sort e insertion sort.
- **palindromo.c**: Verifica se uma string é um palíndromo usando recursão.
- **pilha.c**: Implementação de uma pilha com as operações `push`, `pop`, `peek`, `isEmpty`, `clear` e `printPilha`.
- **struct_parametro.c**: Demonstra a passagem de structs como parâmetro para funções, tanto por valor quanto por referência.
